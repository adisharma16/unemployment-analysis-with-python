# Unemployment in India Analysis

## Overview

This project analyzes unemployment trends in India using Python and data visualization techniques. The goal is to explore unemployment rates across different states, regions, and time periods to gain insights into employment patterns and economic conditions.

## Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Unemployment rate visualization
* State-wise unemployment comparison
* Trend analysis using graphs and charts
* Insights from unemployment data

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Structure

```
Unemployment-in-India/
│
├── Unemployment in India.ipynb

```

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Unemployment-in-India.git
```

2. Navigate to the project directory:

```bash
cd Unemployment-in-India
```

3. Install required packages:

```bash
pip install -r requirements.txt
```

## Usage

Open the Jupyter Notebook and run all cells:

```bash
jupyter notebook
```

Then open:

```
Unemployment in India.ipynb
```
